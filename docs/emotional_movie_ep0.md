# 閉館後のFLATUP — エモーショナル・ムービー正本（第0話「今日のチャンピオン」）

FLATUP GYM の核となるエモーショナル戦略の正本ドキュメント。
「世界一初心者に優しい格闘技ジム」を、説明ではなく**物語**で伝えるシリーズの制作基準書。

- 原案: JIN（2026-07-13）
- この版: 原案をブラッシュアップした v1.0
- 使い方: 「§2 マスタープロンプト」を動画制作AI・制作エージェントにそのまま渡す。§3以降は人間（JIN）向けの実務ガイド。

---

## §1 原案からの主な変更点（ブラッシュアップの根拠）

| # | 変更 | 理由 |
|---|------|------|
| 1 | **冒頭フックを0〜3秒に前倒し**（タイマー起動→グローブの目が開く→「秘密の会議」の字幕） | Reels/Shortsは最初の2秒で離脱が決まる。原案の「暗いジムをゆっくり4秒」は世界観としては正しいが、フィード上では黒い画面＝スキップ対象。「目覚め」を先頭に持ってきて、静けさの提示は3〜7秒に圧縮 |
| 2 | **セリフを9本→6本に削減**、Scene 6の4連セリフを解消 | 日本語音声は1秒あたり約5〜6文字。原案のScene 6（5秒）に4本は物理的に入らない。削ったセリフは字幕・表情に移管 |
| 3 | **「母の手を離す」カットのフレーミング指定を追加** | 手と手の接触・分離はAI動画が最も破綻しやすい描写。手元のクローズアップ単独カットに分離し、全身では描かない |
| 4 | **音声2トラック戦略**（字幕完結版を主軸、声あり版を後追い） | Reelsの多数派は無音視聴。さらにAI子ども声・キャラ声は品質が不安定で、不気味さが出ると感動が全損する。第0話は「字幕＋SE＋BGMのみ」で成立する設計にし、声は品質が確認できてから追加 |
| 5 | **オーガニック版と広告版を明確に分離** | 動物リール運用で確立済みの鉄則「動画内に宣伝を入れると伸びない」を適用。オーガニック版はロゴまで、CTA 3秒は広告版のみ |
| 6 | **制作ツールを既存基盤にマッピング**（Seedance 2.0 / fal.ai ほか） | `npm run reel` のFAL基盤・キャラ固定テク（image-to-video起点）が既にある。ゼロから探さない |
| 7 | **KPIと計測を追加** | 「85点自己評価」は制作品質ゲート。公開後の学習には視聴データが必要（3秒維持率・完走率・保存・LINEタップ） |
| 8 | **字幕のセーフゾーン指定を追加** | 9:16のReels UIはボタン・キャプションで画面の下部約20%と右端が隠れる。字幕は中央〜やや下の安全域に固定 |
| 9 | **尺の整合を明記** | 本編35秒＋広告CTA 3秒＝広告版38秒。15秒版は「切り出し」でなく再設計（§6） |
| 10 | **シリーズ運用ルールを追加**（#閉館後のFLATUP、次回予告、話数表記） | 単発でなく連載として育てる。動物リールで実証済みの「シリーズ化で伸びる」を適用 |

原案の以下は**そのまま維持**（すでに正しい判断のため）:
「ピクサー風」を避けた商用安全なスタイル指定 / 泣かせ演出の禁止 / 一発生成でなく工程分割 / キャラ3体限定 / 静止画正本→毎回参照 / 1カット1動作 / 仮編集ファースト / 5日以内公開優先。

# 完成台本（改訂版・35秒）

セリフは全6本。v1ではすべて字幕表示。

## Scene 1: 目覚め（コールドオープン）
時間: 0〜3秒
映像:
暗いジムの床。タイマーの表示が 00:00 に変わり、小さな電子音。
その音で、床に置かれたグローブが、そっと目を開ける。
カメラ: グローブの顔に寄った固定気味のクローズアップ。
字幕（フック）:
「閉館後のジムで、毎晩ひらかれる——」
「秘密の会議。」
音: タイマーの電子音。革がわずかにきしむ音。
狙い: 最初の2秒で「何が始まるの？」を成立させる。黒画面で始めない。

## Scene 2: 夜のジム
時間: 3〜7秒
映像:
引きの画。夜のFLATUP GYM全景。子どもたちが帰ったあとの静けさ。
時計の秒針だけが響く。グローブが小声で仲間を起こす。
カメラ: ジム全体を見せるゆっくりとした前進（ドリーイン）。
セリフ:
グローブ「みんな、起きて。」（小声。秘密の時間が始まる感覚）
字幕: 「子どもたちが帰ったあと——」
音: 遠くで閉まるドア。秒針。

## Scene 3: 秘密の会議
時間: 7〜13秒
映像:
グローブ、ミット、タイマーが小さな円になって集まる。
背景に、眠っているサンドバッグや縄跳び。
セリフ:
タイマー「今夜の議題。『今日いちばん強くなった子』は？」
音: 革が動く小さな音。控えめで少しユーモラスな木琴の音楽。

## Scene 4: 候補（ミスリード）
時間: 13〜17秒
映像:
昼間の回想。上手な子が力強くミットを打っている。
セリフ:
グローブ「あの、パンチが一番強かった子？」
ミット「ううん。」
演出: 「ううん。」の直後、1拍だけ無音に近づける。ここが物語の転回点。

## Scene 5: 小さな勇気（感情の核・最長シーン）
時間: 17〜26秒
映像（回想・カットを分割する）:
a. 初めて来た小さな女の子。母親の後ろに隠れている。
b. 先生がしゃがみ、女の子と同じ目線で優しくミットを差し出す。
c. 【手元だけのクローズアップ】母親の手から、女の子の小さな手が
   そっと離れる。（このカットに全身や顔を入れない）
d. 女の子が一人で一歩前へ出る。
e. 弱くても、自分の力でミットを一度だけ打つ。
f. 母親は言葉を発さず、少し驚き、静かに微笑む。
セリフ:
ミット「今日、はじめて『ひとりで前に出た』子。」
重要:
・涙を流させない。母親の表情は、寂しさと喜びが同時に存在する繊細なもの
・ミットを打つ音は軽く、控えめに

## Scene 6: 本当のチャンピオン
時間: 26〜31秒
映像:
夜の会議へ戻る。グローブが少し目を潤ませながら笑う。
タイマーの表示が「1」を灯す（チャンピオン決定の演出。文字は使わず数字のみ）。
セリフ:
タイマー「本日のチャンピオン、決定。」
グローブ「ほんの少し。でも、ちゃんと、強くなった。」
演出: ミットは言葉ではなく、頷きと柔らかい表情だけで応える。

# 動画生成工程

・1回で35秒を生成しない。1カット3〜5秒に分割する
・各カットの動きは一つだけに限定する
・生成の起点は必ず正本静止画からの image-to-video とする
・キャラクターの形を維持し、大きなアクションよりも
  小さな視線、呼吸、間を優先する

優先動画化カット（4つ。他は静止画＋ズーム/パン/光/瞬きで表現）:
1. グローブが目を開ける（Scene 1）
2. 手元クローズアップ: 手がそっと離れる（Scene 5c）
3. 女の子が一歩前へ出る（Scene 5d）
4. 朝日が差し込む（Scene 7）

# 編集工程

最初に静止画だけで仮編集（アニマティック）を作り、以下を確認する。
・35秒以内に理解できるか
・無音で見ても伝わるか（最重要。v1の完成条件）
・何の広告か分かるか（オーガニック版は「ジムの物語」と分かれば良い）
・親が自己投影できるか
・泣かせようとしすぎていないか
・（広告版）最後に体験予約へ進めるか

仮編集がOKになってから、優先4カットだけを動画化する。

15秒広告版は35秒版の切り出しではなく再設計する:
Scene 1（目覚め 0〜3秒）→ Scene 5c/5d（手が離れる・一歩前へ 3〜9秒）
→ Scene 7（朝＋ロゴ 9〜12秒）→ CTA（12〜15秒）

## §3 制作ツールマッピング（既存基盤との接続）

| 工程 | 第一候補 | 代替 | 備考 |
|------|---------|------|------|

| 静止画→動画（4カット） | **Seedance 2.0（fal.ai）** — `FAL_KEY` は取得済みの既存基盤 | Veo（物理・光が最強）/ Kling | 動物リールと同じ image-to-video 起点方式。1カット3〜5秒 × 4本 ≒ 数十円〜数百円 |
| BGM | Suno / Udio（商用プラン） | フリー音源（Artlist等） | 「木琴＋ピアノ＋弦、静かな希望」で指定 |

## §4 5日間の制作スケジュール

| 日 | 作業 | 完了条件 |
|----|------|---------|
| Day 1 | 正本4点（夜のジム全景＋キャラ3体設定画）を確定 | JINが「このキャラで行く」と承認 |
| Day 2 | シーン静止画8枚を生成（毎回正本を添付） | キャラ・内装がブレていない |
| Day 3 | 静止画のみで仮編集35秒 → 無音チェック | 無音で見て意味が分かる |
| Day 4 | 優先4カットを動画化 → 差し替え → BGM/SE/字幕 | v1完成 |
| Day 5 | セルフチェック85点 → 予備日/修正 → 公開 | オーガニック版投稿 |

広告版（+CTA 3秒）と15秒版は、オーガニック版の反応を1週間見てから制作してよい。

## §5 KPIと公開後の判断

- **3秒視聴維持率**: 70%未満なら冒頭フック（Scene 1）を差し替えてリテイク
- **完走率**: 40%以上で合格ライン（35秒尺として）
- **保存数・シェア数**: 「親が誰かに送りたくなったか」の代理指標。いいねより重視
- **プロフィール遷移→LINEタップ**: オーガニック版の導線はプロフィールリンクのみ
- 反応が良ければ広告版を出稿し、CTA付き38秒版でLINE予約を計測

## §6 シリーズ運用

- シリーズ名: **閉館後のFLATUP**
- キャプション冒頭に話数を入れる（動物リールで実証済みの連載フォーマット）
- ハッシュタグ: `#閉館後のFLATUP` を固定タグ化。＋ジム系・子育て系タグ
- 各話の最後に静止画1枚の「次回予告」を入れてもよい（第1話以降）

| 話数 | タイトル | 感情の核 |
|------|---------|---------|
| 第0話 | 今日のチャンピオン | 怖くても一歩前へ出る |
| 第1話 | 泣いてもいい | 感情を出すことは弱さではない |
| 第2話 | もう一回 | 失敗しても立ち上がる |
| 第3話 | 負けた日の会議 | 勝敗より挑戦を認める |
| 第4話 | 優しいパンチ | 力は人を守るためにある |
| 第5話 | 誰かを強くする側へ | 先輩になる、教える喜び |

- キャラ3体の正本は全話で使い回す（第0話で作った参照画像がシリーズの資産になる）
- 新キャラ（サンドバッグ、縄跳び等）は1話につき最大1体まで追加

## §7 運用上の絶対ルール

- 投稿・出稿の最終実行は必ずJIN（AI・スタッフは下書きまで）
- 実在の子ども・保護者・スタッフの顔をキャラに再現しない
- 生成画像・動画内に文字を描かせない（字幕・ロゴは編集で後入れ）
- Instagram投稿時はAI生成ラベルON
- 既存アニメ映画のキャラ名・スタイル名（「ピクサー風」等）をプロンプトに書かない
