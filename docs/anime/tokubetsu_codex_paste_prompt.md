■ 作品
FLAT UP GYM（キックボクシング・格闘技ジム）のブランド短編アニメ
タイトル：GYMSTORYs 特別編『そのちから、なにに使う？』
テーマ：子どもに格闘技を習わせるメリット／“力と責任”／だから「どこで・誰に習うか（ジム選び）」が大切。
狙い：子どもには「つよさは、やさしさに使う番」、保護者には「強さが光にも影にもなる。決めるのは習う場所」を、同時に伝える二層構造。

■ 絶対ルール（違反したら作り直し）
- 既存IPを模倣しない。スター・ウォーズ／ジェダイ／“フォース”／ライトセーバー、スパイダーマン等の
  キャラ・造形・固有名・決め台詞・効果音は使用禁止。**剣・銃・光る剣を出さない。**
- 力の表現は独自の「**手に灯る光**」のみ：守るために使う＝あたたかい金色 `#ffcf6a`／
  傷つけるために使う＝冷たい青紫 `#7b6bd6`。この色対比で善悪を3秒で伝える。
- 「大いなる力には大いなる責任が伴う」は“広く知られた格言”として、キャラの子ども言葉で言わせる
  （特定作品の引用として見せない）。
- 暴力を派手・かっこよく見せない。出血・恐怖・大人の威圧なし。子どもの人格否定なし。
  家庭・保護者を責めない。格闘技を危険物として否定しない（問題は「力の使い方を教わらないこと」）。

■ ビジュアル方針
- あたたかい高品質3D風だが、Canvas図形で表現できる範囲の「かわいいパステル・シール調」。
- パレット：ピンク `#ff8fbb`・薄ピンク `#ffd0e0`・クリーム `#fff6ea`・青紫影 `#7b6bd6`・金 `#ffcf6a`・白 `#ffffff`・墨 `#2a2140`。
- 白い床・緑マット・観葉植物・蛍光ピンク/イエローのサンドバッグ、を簡易図形で背景に。

■ 登場（すべて手続き描画のオリジナル・ちびキャラ。外部画像不可）
- ハル（主人公・男の子）：2.5頭身、丸い頭、短い黒髪、素朴なシャツ、小さな赤いグローブ。表情は眉と口で。
- あぷちゃん（導き役の女の子）：2.5頭身、高いポニーテール（濃茶）、大きな丸い瞳、ハート柄シャツ、
  大きな赤グローブ（指は描かない）、裸足。上から教えず、問いで気づかせる。
- 小さい子（からかわれる子）：さらに小さいちび、控えめな色。
- 保護者：顔を描かない後ろ姿シルエット（頭＋肩の丸い塊）のみ。
- ※キャラは丸・楕円・角丸四角の組み合わせで十分。誰かに似せない。

■ 「光」の演出（中核）

  openlight：青紫→金へ戻り、最後に金色が大きくまわりを温める。

■ 技術要件（1ファイル index.html + 書き出しスクリプト）
- 純Canvas + バニラJS。外部CDN/フォント/画像なし（フォントはsystem/Arial系＋丸ゴシック指定）。
- 画面サイズは URL で可変：`?w=1280&h=720`（横）/ `?w=1080&h=1920`（縦）。縦横で文字位置を調整。
- 時刻ベース描画：`draw(t)`（時刻tの1枚）、`window.seek=t=>{draw(t);return true;}`、
  `window.DUR`（全シーン秒の合計）、`window.ready=true`（初期化完了）。
- `?loop=1` でリアルタイムのループ再生（サイト埋め込み用）。
- 予約CTA：Canvasのボタン絵とは別に、**本物のHTML `<a id="booking-cta" target="_blank" rel="noopener noreferrer">`** を用意し、
  loop再生の cta シーン中だけ Canvas のボタン位置に重ねて表示（キーボード/スクリーンリーダー対応）。
  MP4書き出し（loopなし）では非表示。href は上記LINE、押下で開く。
- BGM：任意。付けるなら WebAudio でその場生成（外部音源なし）。静かなシーン（mentor/watch/sunrise/cta）は
  やさしい音、それ以外は軽快に。付けない場合は無音で可。
- コードは読みやすく、各シーンは `SCENES[type](t,lt,p,sc)` の形。未定義typeを作らない。

■ MP4書き出しスクリプト（render.mjs 等）
- Playwright（Chromium）で index.html を開き、`window.ready` を待ち、`seek(t)` を 1/30 秒刻みで進めて
  スクリーンショット→ffmpeg で MP4 化。CLI：`--cut=main|60|30|15` `--vertical` `--out=<file>`。
- 生成コマンド例：
  - `node render.mjs --cut=main`（横）/ `node render.mjs --cut=main --vertical`
  - `node render.mjs --cut=60 --vertical` / `--cut=30 --vertical` / `--cut=15 --vertical`
- ffmpeg が無い環境なら、Playwright の動画録画（webm）にフォールバックしてよい。

無意味な群衆/いじめをかっこよく見せる、を禁止。

- [ ] `?cut=main|60|30|15` で正しく尺・シーンが切替、`window.ready===true` 後に全シーンを数フレーム描いて pageerror 0。
