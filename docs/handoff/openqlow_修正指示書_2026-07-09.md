## 2. 直す不具合（重要度順）

### 🔴 重大1: レディースが「土曜14:00」と回答（正: 14:30）
**再現**（IMG_3850/3851）: 「レディース」→「土曜**14:00**のクラスです」。「14:30では？」と訂正しても
「レディースクラスは土曜**14:00**となっております」と**誤情報を断言し訂正拒否**。クイックリプライも「🙋‍♀️ 14時 レディース」。

**期待**: 「レディースクラスは土曜 **14:30** です」。

**openqlow での探し方 / 直し方**:
- grep: `14:00` / `14時` / `1400` / `14：00` / `レディース` / `ladies`
- canon（`src/shared/canon.ts` / `port/aika/flatup_canon.ts`）の `scheduleLadies` が **14:30 か** を確認（PR #40 反映済みか）。
- **canon 以外にベタ書きされた 14:00** を探す（濃厚）:
  - クイックリプライのボタン **label / postback**（例「14時 レディース」→「14:30 レディース」）
  - 体験予約フローの**クラス時間テーブル / 固定文言テンプレ**
- ※本番が14:00＝canon未デプロイ or 別箇所ベタ書きの両方を確認。直したら**再デプロイ**。

### 🔴 重大2: キッズ体験のクイックリプライがクラス時間と噛み合わない
**再現**（IMG_3840/3848/3849）: 「火曜か木曜どちら？」と**曜日を聞く場面**なのに、ボタンが
「🕙 10〜12時」「🕕 18〜20時」＝**時間帯ボタン**。さらにキッズに無い「10〜12時」を選ばせ、選ぶと
「キッズクラスは火木18:00・土13:00のみです」と**拒否**。

**期待**:
- キッズ体験の時間候補は**実クラスのみ**＝火木 **18:00**、土 **13:00**。「10〜12時」は出さない（在館時間でありキッズクラスではない）。
- **曜日を聞く場面は曜日ボタン（火曜 / 木曜 / 土曜）**を出す。時間帯ボタンを出さない。

**openqlow での探し方 / 直し方**:
- grep: `quickReply` / `items` / `postback` / `label` / `action`
- 「10〜12時」「18〜20時」を出す分岐が**種目（キッズ/レディース/大人）に関係なく共通の在館時間**を出していないか。
  → **種目ごとに候補時間を分ける**。キッズは {火:18:00, 木:18:00, 土:13:00} のみ。
- 曜日確認ステップと時間確認ステップで**どちらを聞いているか**の状態を持ち、場面に合うボタンだけ出す。

### 🔴 重大3: 予約フローが曜日を保持できずループ
**再現**（IMG_3841/3849）: 「火曜か木曜どちら？」→ ユーザーが答えても時間確認へ戻る堂々巡り。
「火曜日」「火曜日火曜日」と連打しても進まない（曜日を認識・保存できない）。

**期待**: 確定した「曜日」「時間帯」を**状態として保持**し、**両方揃ったら名前確認へ進む**。
自由入力の曜日（"火曜日" 等）も postback と同じ扱いで曜日スロットに束縛する。

**openqlow での探し方 / 直し方**:
- grep: `slot` / `state` / `step` / `session` / `reservation` / `予約` / `体験` / `booking` / `trial` / `context` / `名前` / `name`
- 予約フローのステートマシン（現在ステップ・slot filling）を確認。
- **時間スロットと曜日スロットが互いを上書きしていないか**（上書き→無限ループの典型）。別スロットとして独立保存する。
- ユーザーテキスト "火曜日" を曜日スロットに束縛するパーサを追加/修正。
- 曜日 + 時間が揃ったら次ステップ（名前確認）へ遷移させる。

> 重大2と重大3は**同じ予約フロー**なのでセットで直すのが自然。

### 🟡 中4: 入会確認【1/2】に健康・飲酒・怪我の注意が無い
**観察**（IMG_3842/3853）: デプロイ版の【1/2】重要事項は「1. 退会・休会 / 2. キャンペーン入会」の**2点のみ**。
正本では ③健康状態（高血圧・心疾患・脳血管疾患・てんかん等）④飲酒時利用禁止 ⑤怪我の自己責任・保険案内 も入れる約束。
オーナーが体調・怪我リスクを懸念（安全・法務リスク）。

**期待**: 同意文テンプレに ③④⑤ を追記し、正本の全5項に揃える。

**openqlow での探し方**: grep `重要事項` / `1/2` / `キャンペーン入会` / `違約金` / `退会` で同意文テンプレを特定。

**貼り付け用（正本§24全文・`flatup-ai-os/src/data/line_flows_100.md:135-156`）**:
```
【1/2】ご入会前のかんたん確認☀
ご入会ありがとうございます！
登録の前に、大事なことだけご確認ください。
① 退会・休会について
退会・休会は、申請した月の翌月末で完了となります。
そのため、翌月分までは通常会費が発生します。
例：7月に申請した場合、8月末で退会・休会となります。
② キャンペーン入会について
キャンペーンでご入会の方は、入会日から1年未満で退会された場合、入会金相当10,000円が発生する可能性があります。
1年続けていただければ、かかりません。
③ 健康状態について
高血圧、心疾患、脳血管疾患、てんかん、その他医師から運動制限を受けている方、体調に不安がある方は、事前に医師へ相談のうえご利用ください。
体調不良、めまい、息苦しさ、強い疲労感、痛みがある場合は、運動をお控えください。
④ 飲酒時の利用について
飲酒時、または酒気を帯びた状態でのジム利用・練習参加はできません。
⑤ 怪我について
スパーリング、対人練習、ミット打ち、各種トレーニングには怪我の可能性があります。
練習中の怪我は自己責任となりますので、万が一に備えてスポーツ保険等への加入もご検討ください。
※このLINEは要約です。詳細は入会規約に準じます。
```
> ①退会・②キャンペーン（10,000円・**断定しない**）の文言は**変えない**。③④⑤を**足すだけ**。
