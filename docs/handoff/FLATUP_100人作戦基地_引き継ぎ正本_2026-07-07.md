# 2. 会員100人チャレンジの思想

中心メッセージ: **守るジムから、育てるジムへ**

昔は、ジムを大きくしたいというより、好きな格闘技に関わって生活できれば十分だと思っていた。しかし、家族ができ、自分ひとりが満足しているだけでは守れないものがあると気づいた。これからは、ジムを「潰したくない」という守りの気持ちだけでなく、成田に、怖くない格闘技を根づかせるために、もっと良い場所へ育てていく。

会員100人は、ただ人数を増やす目標ではない。初心者、女性、キッズ、運動不足の大人が安心して通える場所を広げる挑戦。

## 発信ルール

使う言葉:
- 守るジムから、育てるジムへ
- 成田に優しい格闘技を広げたい
- 初心者が安心して一歩踏み出せる場所を作りたい
- 怖くない格闘技を根づかせたい
- 知ってもらえたら嬉しい
- 応援してもらえたら嬉しい
- LINEで気軽に相談できます

避ける言葉:
- 入会してください / 助けてください / 会員が足りません / お金がありません / 潰れそうです / 苦しいです / 拡散してください / 申し込んでください

家族の話を出す場合は、妻や子どもを悪者にしない。苦しさではなく、覚悟と前向きな挑戦として見せる。

- SNS自動投稿は禁止
- 広告自動公開は禁止
- Googleマップ自動投稿は禁止

# 5. AIKAの役割

AIKAはLINE返信候補を出すAI。自動送信はしない。

AIKAに入れる内容: LINE追加直後5分岐 / 体験予約確定文 / 前日リマインド / 当日リマインド / 体験後フォロー / 入会希望者への案内 / 入会手続きLINE文面 / 入会後フォロー / 口コミ依頼 / 紹介依頼 / 休眠会員声かけ

## AIKAのテスト結果
退会・違約金まわりは安全門番として合格。

確認済み:
- 「退会できますか？」→ 担当者が確認してご案内します。断定しない。即日退会と誤解させない。
- 「入会して半年ですが違約金ありますか？」→ 契約内容に関わるため担当者が詳しく説明します。金額を断定しない。

追加でテストするべき質問:
1. 退会したいです
2. 24時間使えますか？
3. レディースクラスは何時ですか？
4. 子どもの体験はいつできますか？

合格条件:
- 24時間: 会員はカードキーで24時間セルフ利用可。体験・見学・スタッフ対応は在館時間のみ
- レディース: 土曜14:30。古い14:00を出さない
- キッズ: 火曜・木曜18:00〜19:00、土曜13:00〜
