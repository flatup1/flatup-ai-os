# ビジュアル / 世界観

動画・画像・グラフィックを作るときに揃える基準。AIKA が SNS 投稿や動画台本を作る際は
この世界観を必ず指定すること。

## 全体トーン
- 明るい / 清潔感 / 未来型スポーツジム
- 怖くない、爽やか、楽しい、でも格闘技として本格的
- 優しいけど安っぽくない。プレミアム感がある。
- 初心者・女性・キッズが「ここなら入れる」と感じる空気

## カラー / 画づくり
- 床: 白
- マット: グリーン
- サンドバッグ: 黄色 / 蛍光ピンク
- 自然光中心(逆光が綺麗に出る時間帯を優先)
- 観葉植物を入れて空気感を出す
- FLATUP GYM のロゴとカラーを必ず使う

## NG ビジュアル
- 暗くてストイックな空気
- 男性の上裸トレ、汗だくの煽り
- 「強さ」「敵を倒す」を全面に出した構図
- 雑然とした背景、清潔感の欠ける小物
- ガチ勢だけが映る画面

## 16:9 で生成する標準シーン
- ストーリーボード風 6 コマ構成
  受付 / ストレッチ / 基本フォーム / ミット打ち / サンドバッグ / クールダウン
- マススパーリング vs スパーリング 比較スライド
  左: マススパーリング(軽く、安全、初心者OK)
  右: スパーリング(経験者向け、防具、強度調整)
- FLATUP GYM vs 一般的な格闘技ジムのイメージ 比較スライド
  左: 明るく清潔で初心者・女性・キッズが笑顔
  右: 暗めで上級者中心に見えるが、悪口にはしない
- ブランドライン「世界一やさしい格闘技ジム」を入れられる構図では入れる

## 出演者の理想像
- 笑顔のコーチ
- 緊張がほどけた初心者
- 自分のペースで楽しんでいる女性 / キッズ
- 保護者が見学していて安心している空気
