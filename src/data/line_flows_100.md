# LINE文面正本（会員100人作戦基地・2026-07-07版）

このファイルは、LINE公式アカウントで使う文面の正本。
AIKAは返信候補を作るとき、この文面と数字（時間・料金）を正として使う。
AIKAは自動送信しない。送信は必ず人間が行う。

クラス時間の正本:
- キッズクラス: 火曜・木曜 18:00〜19:00 / 土曜 13:00〜
- レディースクラス: 土曜 14:30〜（以前の「土曜14:00」は古い情報。絶対に使わない）
- 会員はカードキーで24時間セルフ利用可。体験・見学・スタッフ対応はスタッフ在館時間のみ（平日10:00〜12:00 / 18:00〜20:00、土曜クラス制、日曜・祝日は原則体験不可）
